Pension vocabulary

Version: 0.10

Pension
Information about a person's pension.
endDate
The last date when a temporary pension benefit will be paid. Optional. Only exists for temporary pension benefits.
startDate
The date when the pension is paid to the beneficiary for the first time.
provisional
Boolean value, set to true if the pension decision is not confimed yet. true = provisional, false = confirmed. Optional. Default value false.
typeCode
Short code representing the type of the pension benefit. KAEL = kansaneläke, TAKE = takuueläke, KUKI = kuntoutustuki, TKEL = pysyvä työkyvyttömyyseläke.
typeName
Human-readable type of the pension benefit. Can be localized to the credential holder's language. Possible values:
typeCode fi sv en
KAEL Kansaneläke Folkpension National pension
TAKE Takuueläke Garantipension Guarantee pension
KUKI Kuntoutustuki Rehabiliteringsstöd Cash rehabilitation benefit
TKEL Pysyvä työkyvyttömyyseläke Sjukpension Disability pension
